# Logo Guidelines

## Overview

The logo is a critical element that represents the brand's identity. Follow the appropriate format and guidelines depending on the medium and context in which the logo is used.

## Usage Guidelines by Medium

### Usage in Video

When using the logo in video content, follow these guidelines:

| Item | Recommendation |
|------|----------------|
| Display duration | Ensure at least 2 seconds of visibility |
| Position | Maintain sufficient margin from the screen edges |
| Size | Use at or above the minimum size that ensures legibility |
| Animation | Keep motion subtle to avoid undermining the brand image |
| Background | Choose a background color that provides enough contrast for the logo to be clearly visible |

#### Notes by Video Format

- **Video ads**: Display the logo at the beginning or end
- **Product introduction videos**: Use as a watermark sparingly
- **Social media videos**: Consider logo placement suited to square or vertical formats

### Usage in Audio

When using audio content or a sound logo, follow these guidelines:

| Item | Recommendation |
|------|----------------|
| Duration | A sound logo of approximately 3 to 5 seconds is recommended |
| Volume | Adjust to a level that harmonizes with the overall content |
| Timing | Place at the beginning or end of the content |
| Tone | Use a timbre and melody consistent with the brand image |

#### Audio Usage Scenarios

- **Podcasts**: Use in intros and outros
- **Video content**: Combine with the visual logo
- **Audio ads**: Sound branding for recognition

## Notes for Each Medium

### Prohibited Practices

- Distorting or warping the logo
- Using colors not specified in the brand guidelines
- Using the logo at low resolution
- Reducing visibility due to insufficient contrast with the background
- Placing other elements too close to the logo

### Recommended Practices

- Always use the original logo file
- Maintain adequate clear space around the logo
- Choose the appropriate file format for the intended use:
  - Web: SVG or PNG
  - Print: AI, EPS, or high-resolution PDF
  - Video: Transparent PNG or video format with alpha channel
